Terraform vCloud provider

Terraform vCloud provider is an extension to Terraform for vCloud.

  • Linux
  • Mac
  • Windows

Setup

  1. Download the provider for your machine above
  2. Place it according to terraforms third party documentation
  3. Rename the file to terraform-provider-vcd
  4. Run terraform init in your folder containing the vcd provider
